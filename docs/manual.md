# CPK Analyzer 사용 매뉴얼

---

## 1. 프로그램 실행

### Windows (exe)
1. `CPK_Analyzer` 폴더에서 `CPK_Analyzer.exe`를 더블클릭합니다.
2. 검은 콘솔 창이 뜨고 아래 메시지가 표시됩니다:
   ```
   ==================================================
     CPK Analyzer 시작 중...
     브라우저가 자동으로 열립니다.
     이 창을 닫지 마세요!
   ==================================================
   ```
3. 약 5초 후 브라우저가 자동으로 열립니다.
4. **주의: 검은 콘솔 창을 닫으면 프로그램이 종료됩니다.**

### Mac / Linux (개발 환경)
```bash
streamlit run app.py
```

## 2. 엑셀 파일 업로드

### Step 1: 파일 선택
- 좌측 사이드바에서 **"엑셀 파일 업로드"** 영역을 클릭합니다.
- `.xlsx` 또는 `.xls` 파일을 선택합니다.
- 드래그 앤 드롭도 가능합니다.

### Step 2: 시트 선택
- 파일 업로드 후 **"시트 선택"** 드롭다운이 나타납니다.
- CPK 데이터가 있는 시트를 선택합니다.
- 예: `11. 출력 특성 Cpk`

### Step 3: 그래프 배치 선택
- **1열**: 그래프를 크게 한 줄에 하나씩 표시
- **2열** (기본): 한 화면에 2개씩 나란히 표시

### Step 4: 분석 시작
- **"분석 시작"** 버튼을 클릭합니다.

## 3. 분석 결과 확인

### 3.1 요약 카드
화면 상단에 전체 현황이 표시됩니다:

| 전체 항목 | PASS | FAIL |
|-----------|------|------|
| 9개       | 9개  | 0개  |

### 3.2 요약 테이블
모든 테스트 항목의 주요 지표가 한눈에 보입니다:

| 테스트 | N | 평균 | Cp | Cpk | Pp | Ppk | 판정 |
|--------|---|------|----|----|----|----|------|
| Line Regulation ... | 200 | 13.8460 | 4.49 | 3.44 | 4.49 | 3.45 | PASS |
| Load Regulation ... | 200 | 13.9240 | 4.77 | 4.22 | 4.77 | 4.22 | PASS |

- **PASS**: Cpk >= 1.33 (초록색)
- **FAIL**: Cpk < 1.33 (빨간색)

### 3.3 건너뛴 항목
Spec 값이 없거나 데이터가 부족한 항목은 접을 수 있는 알림으로 표시됩니다:
```
▶ 건너뛴 항목 (1개)
   ⚠ Discharge Test (Spec 변환 실패: max=N/A, min=N/A)
```

### 3.4 공정능력 차트
각 테스트 항목별로 미니탭과 동일한 스타일의 차트가 생성됩니다.

차트 구성:
```
┌──────────┬──────────────────────┬─────────────────────┐
│          │        LSL  Target USL│                     │
│ Process  │   ▓▓▓▓                │ Potential (Within)  │
│ Data     │  ▓▓▓▓▓▓▓▓            │  Capability         │
│          │ ▓▓▓▓▓▓▓▓▓▓           │                     │
│ LSL      │▓▓▓▓▓▓▓▓▓▓▓▓          │ Cp   4.49           │
│ Target   │ ▓▓▓▓▓▓▓▓▓            │ CPL  3.44           │
│ USL      │  ▓▓▓▓▓▓              │ CPU  5.54           │
│ Mean     │   ▓▓▓                 │ Cpk  3.44           │
│ N        │                      │                     │
│ StDev    │  — Within (빨간실선)   │ Overall Capability  │
│          │  -- Overall (검정점선) │ Pp   4.49           │
│          │                      │ ...                 │
├──────────┴────────┬─────────────┴─────────────────────┤
│ Observed Perf.    │ Exp. Within    │ Exp. Overall      │
│ PPM < LSL  0.00   │ PPM < LSL 0.00 │ PPM < LSL  0.00   │
│ PPM > USL  0.00   │ PPM > USL 0.00 │ PPM > USL  0.00   │
│ PPM Total  0.00   │ PPM Total 0.00 │ PPM Total  0.00   │
└───────────────────┴────────────────┴───────────────────┘
```

## 4. 차트 복사 (보고서 붙여넣기)

### 방법
1. 원하는 차트 이미지를 **클릭**합니다.
2. 이미지 테두리가 **초록색**으로 바뀌면 복사 완료입니다.
3. 보고서(Word, PowerPoint 등)에서 **Ctrl+V**로 붙여넣습니다.

### 지원 브라우저
- Chrome (권장)
- Edge
- Firefox (지원 안 될 수 있음)

### 복사가 안 되는 경우
- "복사 실패" 알림이 뜨면 Chrome 또는 Edge로 시도해주세요.
- 또는 차트를 **우클릭 → 이미지를 다른 이름으로 저장**으로 대체할 수 있습니다.

## 5. 엑셀 파일 준비 가이드

프로그램이 자동으로 데이터를 인식하려면 아래 형식을 따라야 합니다.

### 필수 요소

| 요소 | 설명 | 예시 |
|------|------|------|
| 측정항목 행 | "측정항목" 텍스트가 포함된 셀 | A1: `측정항목` |
| Spec max 행 | "Spec max" 또는 "max" | A2: `Spec max` |
| Spec min 행 | "Spec min" 또는 "min" | A3: `Spec min` |
| 테스트 항목명 | 측정항목 행의 각 열 | B1: `Line Regulation 90V47Hz` |
| USL / LSL 값 | 숫자 (각 테스트 항목 열) | B2: `14.66`, B3: `13.34` |
| 시리얼번호 | 10자 이상, 영문+숫자 조합 | A12: `A10024MVT001` |
| 데이터 | 숫자 값 | B12: `13.846` |
| Result 열 | "Result" 텍스트 (선택) | K1: `Result` |

### 예시 레이아웃
```
     |    A         |      B          |      C          | ... |    K
─────+──────────────+─────────────────+─────────────────+─────+─────────
  1  | 측정항목      | Line Reg 90V    | Load Reg 230V   | ... | Result
  2  | Spec max     | 14.66           | 14.66           | ... |
  3  | Spec min     | 13.34           | 13.34           | ... |
 4~11| (기타 정보 — 무시됨)                                     |
  12 | A10024MVT001 | 13.846          | 13.924          | ... | PASS
  13 | A10024MVT002 | 13.851          | 13.920          | ... | PASS
 ... | ...          | ...             | ...             | ... | ...
 211 | A10024MVT200 | 13.842          | 13.918          | ... | PASS
```

### 주의사항
- Spec max/min 값은 반드시 **숫자**여야 합니다 (텍스트 "N/A" 등은 건너뜀).
- 데이터 영역에 빈 셀이 있어도 괜찮습니다 (자동으로 건너뜀).
- 시트 내 기존에 계산된 CP/CPK 값은 무시하고, 원본 데이터만 사용합니다.

## 6. 용어 설명

| 용어 | 의미 |
|------|------|
| **LSL** | Lower Specification Limit — 규격 하한 |
| **USL** | Upper Specification Limit — 규격 상한 |
| **Target** | 목표값 = (USL + LSL) / 2 |
| **Cp** | 잠재 공정능력 — 규격 폭 대비 공정 변동 |
| **Cpk** | 실제 공정능력 — 규격 중심 이탈 반영 |
| **Pp** | 전체 공정능력 (장기 변동 기반) |
| **Ppk** | 전체 공정능력 (장기 + 중심 이탈 반영) |
| **Cpm** | 목표값 대비 공정능력 |
| **PPM** | Parts Per Million — 백만 개당 불량 수 |
| **Within StDev** | 군내 표준편차 — 단기 변동 |
| **Overall StDev** | 전체 표준편차 — 장기 변동 |

### 판정 기준
| Cpk 값 | 판정 | 의미 |
|--------|------|------|
| >= 1.33 | **PASS** | 공정능력 충분 |
| 1.00 ~ 1.33 | 주의 | 개선 필요 |
| < 1.00 | **FAIL** | 공정능력 부족 |

## 7. 문제 해결 (FAQ)

### Q1: exe를 실행했는데 "액세스할 수 없습니다" 에러가 나요.
**A:** Windows 보안이 차단한 것입니다.
1. 다운로드한 ZIP 파일 우클릭 → 속성
2. 하단 "차단 해제" 체크 → 확인
3. 그 다음 압축 풀기

### Q2: 브라우저가 안 열려요.
**A:** 콘솔 창에서 5초 기다려보세요. 안 열리면 직접 브라우저에서 `http://localhost:8501` 입력.

### Q3: 특정 테스트 항목이 빠져있어요.
**A:** "건너뛴 항목" 알림을 확인하세요. 보통 아래 원인입니다:
- Spec max/min 값이 숫자가 아님 (예: "N/A", 빈 셀)
- 데이터가 2개 미만

### Q4: 차트 클릭해도 복사가 안 돼요.
**A:** Chrome 또는 Edge 브라우저를 사용해주세요. Firefox는 Clipboard API 지원이 제한적입니다.

### Q5: 그래프가 미니탭과 약간 다르게 보여요.
**A:** 계산 결과(수치)는 미니탭과 정확히 동일합니다. 히스토그램의 bin 개수나 곡선 모양이 약간 다를 수 있지만, 이는 시각적 차이일 뿐 분석 결과에 영향을 주지 않습니다.

### Q6: 콘솔 창에 에러가 나와요.
**A:** 에러 내용을 캡쳐해서 개발자에게 전달해주세요. 콘솔 창은 에러 확인 후 Enter를 누르면 닫힙니다.

### Q7: 중국에서도 사용할 수 있나요?
**A:** 네. 이 프로그램은 인터넷 연결 없이 로컬에서 동작합니다. exe 파일을 이메일이나 USB로 전달하면 됩니다. Chrome이나 Edge 브라우저만 있으면 됩니다.
